
「软萌Q版·糖果治愈风」

一、最终定版：整体美术风格

风格名称

软萌Q版·糖果治愈风（Flat Soft Cute）

一句话定位

圆润、无棱角、高饱和低对比、超可爱、玩家看了就想点。

为什么选它

• 微信小游戏留存最高的风格

• 合成/宠物/养成类最吸量

• 美术成本低、迭代快、不挑画师

• 适配所有年龄：小孩喜欢、女生喜欢、男生不反感
二、角色造型规范（所有宠物统一）

1. 体型比例（统一模板）

• 头身比：2头身～2.5头身
头大、身体小、腿短、无脖子

• 所有边缘必须圆角
禁止尖角、尖刺、硬棱角

• 眼睛：超大圆眼，占脸 1/3

• 表情统一：呆萌 / 微笑 / 治愈

2. 线条规范

• 线条：统一宽度黑色轮廓线

• 线宽：8–12px（统一不变）

• 不做复杂排线、不做渐变线

3. 上色规范（超级简单）

• 平涂为主 + 1层软阴影

• 不画复杂高光

• 不做脏色、不做写实光影

• 色彩：明亮、干净、糖果色

4. 分辨率（小程序最优）

• 宠物单只：512×512px

• 图标：128×128px

• 背景：750×1334px（竖版）

• 全部导出 PNG 透明底
三、色彩体系（8大元素统一配色）

• 火：#FF5A36 红橙

• 水：#42B9FF 天蓝

• 草：#56E860 草绿

• 雷：#9966FF 紫蓝

• 土：#D8AA5E 土黄

• 风：#A8E6FF 浅青

• 光：#FFFADE 米黄亮

• 影：#6B6B8B 灰紫

所有宠物只从这8套色里取主要配色，但是不要只用这个颜色，还需要有每个宠物的特点。
四、7大种族外观规范（美术直接照做）

1. 兽灵族（狐、猫、兔、熊）

• 毛茸茸质感

• 圆耳朵、短手脚

• 尾巴大、蓬松

2. 羽翼族（鸟、蝶、蜂）

• 翅膀大、身体小

• 翅膀用简单色块，不画复杂纹路

3. 水族（鱼、龟、水母）

• 身体半透明感

• 泡泡、水纹简单点缀

4. 植灵族（花、草、蘑菇）

• 叶子圆润

• 花朵大、表情呆萌

5. 元素族（灵体）

• 雾状、半透明

• 边缘模糊、发光

6. 幻灵族（精灵、幽灵）

• 无脚、漂浮

• 弱发光、柔和

7. 龙族（重点！）

• 角短、圆钝

• 身体粗短、不凶猛

• 翅膀圆润、可爱优先
五、界面UI风格（配套统一）

• 全部圆角：按钮圆角 20px+

• 弹窗：白色+浅阴影

• 边框：细、柔和

• 字体：粗圆体（禁用宋体、黑体）

• 技能图标：简单几何+元素色，不复杂
六、特效风格（合成/升级/战斗）

• 光效：柔和圆形光斑

• 无刺眼强光

• 粒子：小圆点、小星星、小爱心

• 合成特效：冒泡→发光→诞生
