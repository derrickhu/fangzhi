> 宠物全清单 + 属性 + 种族 + 元素 + 克制 + 合成配方 + 培养 + 技能 + 队长技
> 可直接交付程序 & 美术
>
> **v3 核心改进**：彻底解决动物同质化问题，103只基础宠涵盖80+种不同动物/生物形态，杜绝"五只狐狸"式重复。合成宠走全新物种融合路线，每只都有独特辨识度。

## 一、基础世界观与核心规则

### 1. 核心玩法
捕捉宠物 → 固定配方合成（无配方随机兜底）→ 升级升星培养 → 解锁专属技能 / 队长技 → 图鉴收集

### 2. 7 大种族
兽灵族、羽翼族、水族、植灵族、元素族、幻灵族、龙族

### 3. 元素系统

#### 3.1 宠物属性元素（9 种）
火、水、草、雷、土、风、光、影、心

#### 3.2 战斗棋盘珠子（双棋盘制，各 6 种）

根据敌人属性自动切换棋盘类型：

| 棋盘 | 珠子类型 | 对应敌人属性 |
|------|---------|-------------|
| **炎海棋盘** | 火、水、草、光、影、心 | 火 / 水 / 草 / 光属性怪 |
| **岚雷棋盘** | 风、雷、土、光、影、心 | 风 / 雷 / 土 / 影属性怪 |

> 两套棋盘共享：光、影、心三种珠子
> 心珠在两套棋盘中均为回复珠

#### 3.3 克制关系

**炎海棋盘内克制（三角克制）：**
火 → 草 → 水 → 火
光 ↔ 影（互相克制）

**岚雷棋盘内克制（三角克制）：**
雷 → 风 → 土 → 雷
光 ↔ 影（互相克制）

> 克制倍率 ×2.5，被克制倍率 ×0.5
> 心属性对所有属性无克制/被克制

#### 3.4 宠物属性 → 棋盘珠子映射

每只宠物根据自身属性消除对应珠子触发攻击：

| 宠物属性 | 触发珠子 | 所在棋盘 |
|---------|---------|---------|
| 火 | 火珠 | 炎海棋盘 |
| 水 | 水珠 | 炎海棋盘 |
| 草 | 草珠 | 炎海棋盘 |
| 风 | 风珠 | 岚雷棋盘 |
| 雷 | 雷珠 | 岚雷棋盘 |
| 土 | 土珠 | 岚雷棋盘 |
| 光 | 光珠 | 两套棋盘通用 |
| 影 | 影珠 | 两套棋盘通用 |
| 心 | 心珠（回复） | 两套棋盘通用 |

> 当编队中有不匹配当前棋盘的宠物时（如炎海棋盘带了雷属性宠），该宠物无法通过消珠触发普攻，但技能仍可正常释放

### 4. 稀有度（3 档）
| 稀有度 | 代码标识 | 颜色 | 说明 |
|-------|---------|------|------|
| R（幼兽） | T3 | 蓝 #4dabff | 基础宠物 |
| SR（灵兽） | T2 | 紫 #b388ff | 中阶宠物 |
| SSR（神兽） | T1 | 红 #ff5040 | 顶级宠物 |

### 5. 合成总规则
- 任意两只宠物均可合成
- 有配方 → 必出固定宠物
- 无配方 → 按种族 + 元素 + 品质随机产出
- 合成后素材宠消失
- 不公示配方，全靠探索

## 二、宠物成长体系（全宠通用）

### 1. 等级系统
- 等级上限由星级决定
- 升级消耗：经验珠（小 +100 / 中 +500 / 大 +2000，挂机 / 战斗 / 广告产出）
- 升级提升：生命、攻击、回复（每级 +3% 基础属性）
- 经验曲线：`floor(20 + (level-1) × 8 + (level-1)^1.5 × 2)`

### 2. 星级成长线（4 星制）
| 星级 | 等级上限 | 解锁内容 | ATK 倍率 |
|------|---------|---------|---------|
| ★1 | Lv.30 | 无主动技能 | ×1.0 |
| ★2 | Lv.50 | 专属技能 | ×1.3 |
| ★3 | Lv.80 | 技能强化 | ×1.69 |
| ★4 | Lv.100 | 终极形态 + 队长技 | ×2.2 |

### 3. 升星消耗
| 升星 | 条件 |
|------|------|
| 1→2 星 | Lv.30 + 同宠 1 只 |
| 2→3 星 | Lv.50 + 同宠 2 只 |
| 3→4 星 | Lv.80 + 同宠 3 只 + 突破石 |

### 4. 基础属性公式
- HP = ATK × 12
- REC = ATK × 0.4（取整）
- 最终属性 = 基础值 × (1.3^(star-1)) × (1 + (level-1) × 0.03)

### 5. 队长技（★4 解锁，替代原法宝系统）
将宠物设为编队队长（第一位）时，队长技全队生效。每只 SSR/SR 宠物都有独特队长技，R 级宠物队长技为通用弱效果。

#### 5.1 队长技分类
| 类型 | 效果示例 |
|------|---------|
| 攻击增伤 | 全队攻击力 +15%、指定属性伤害 +35% |
| 暴击强化 | 暴击率 +25%、暴击伤害 +40% |
| 防御减伤 | 受到所有伤害 -15%、指定属性伤害 -30% |
| 回血治疗 | 每回合自动回血 5%、消除指定珠回血 5% |
| 护盾血量 | 血量上限 +25%、消除珠获得护盾 |
| 珠率提升 | 指定属性珠出现概率大幅提升 |
| 特殊触发 | 斩杀（残血直接击杀）、反弹伤害、中毒概率 |
| 免疫类 | 免疫眩晕、免疫持续伤害、免疫负面效果 |
| 转珠辅助 | 转珠时间 +2 秒 |

## 三、宠物完整清单

> **设计原则**：
> 1. 每个种族内的基础宠使用**完全不同的动物/生物种类**，杜绝重复
> 2. 同种族 R 级宠覆盖该族核心元素，SR 级宠覆盖进阶元素，SSR 级宠为种族镇族之宝
> 3. 每只宠物外观、性格、战斗风格都有明确差异化
> 4. 合成宠为两种生物的"灵魂融合体"，产生全新物种形态
